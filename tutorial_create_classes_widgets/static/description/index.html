This app is a tutorial in the form of code. The app will learn you how to create, add and use JavaScript files in Odoo.

Learn how to create JavaScript classes and widgets

This app contains all the code you need to create JS classes and widgets.
All the code is documented inline with notes and extra information to understand how to create your own classes and reusable widgets.

Contains

  • New app on the main screen named 'Tutorial: Quiz'
  • New client action 'Tutorial Quiz' with a new view.
  • Quiz data which is loaded/used and checked in the JavaScript side
  • Three new JavaScript widgets named "QuizAction", "QuizSelection" and "QuizSummary". All these widgets create another view. An overview screen, a question screen and a conclusion screen.
  • Inline code explaining how the functionality works.
